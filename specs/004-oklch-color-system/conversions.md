## Removed Tokens

The following `-focus` tokens are removed and replaced with `color-mix()`:

| Removed Token | Replacement |
|---------------|-------------|
| --color-primary-focus | `color-mix(in oklch, var(--color-primary), black 10%)` |
| --color-secondary-focus | `color-mix(in oklch, var(--color-secondary), black 10%)` |
| --color-tertiary-focus | `color-mix(in oklch, var(--color-tertiary), black 10%)` |

## OKLCH Format Notes

**Format**: `oklch(L% C H)`
- **L**: Lightness (0% = black, 100% = white)
- **C**: Chroma (0 = gray, 0.4+ = very saturated)
- **H**: Hue (0-360 degrees)

**Color-mix patterns**:
```css
/* Hover (darken 10%) */
color-mix(in oklch, var(--color-primary), black 10%)

/* Focus (darken 15%) */
color-mix(in oklch, var(--color-primary), black 15%)

/* Disabled (lighten 30%) */
color-mix(in oklch, var(--color-primary), white 30%)
```
