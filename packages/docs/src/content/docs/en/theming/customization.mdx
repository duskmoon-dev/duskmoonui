# Theme Customization

Learn how to create custom themes that match your brand identity while maintaining Material Design 3 principles.

## Creating a Custom Theme

### Basic Custom Theme

Define a custom theme by providing color values in HSL format:

```javascript
// tailwind.config.js
const { duskmoonui } = require('@duskmoon-dev/core');

module.exports = {
  plugins: [
    duskmoonui({
      themes: [
        'sunshine',  // Keep built-in themes
        'moonlight',
        {
          name: 'brand',
          // Primary colors (main brand color)
          primary: '220 90% 50%',              // Blue
          'primary-focus': '220 90% 40%',
          'primary-content': '0 0% 100%',
          'primary-container': '220 90% 90%',
          'on-primary-container': '220 90% 10%',

          // Secondary colors
          secondary: '280 70% 55%',            // Purple
          'secondary-focus': '280 70% 45%',
          'secondary-content': '0 0% 100%',
          'secondary-container': '280 70% 90%',
          'on-secondary-container': '280 70% 10%',

          // Tertiary colors
          tertiary: '160 70% 45%',             // Teal
          'tertiary-focus': '160 70% 35%',
          'tertiary-content': '0 0% 100%',
          'tertiary-container': '160 70% 90%',
          'on-tertiary-container': '160 70% 10%',

          // Surface colors
          surface: '0 0% 100%',
          'surface-dim': '0 0% 87%',
          'surface-bright': '0 0% 100%',
          'surface-container-lowest': '0 0% 100%',
          'surface-container-low': '0 0% 96%',
          'surface-container': '0 0% 94%',
          'surface-container-high': '0 0% 92%',
          'surface-container-highest': '0 0% 90%',
          'on-surface': '0 0% 10%',
          'on-surface-variant': '0 0% 30%',
          'surface-variant': '220 20% 90%',

          // Outline colors
          outline: '0 0% 50%',
          'outline-variant': '0 0% 80%',

          // Semantic colors
          info: '200 90% 50%',
          'on-info': '0 0% 100%',
          'info-container': '200 90% 90%',
          'on-info-container': '200 90% 10%',

          success: '140 70% 45%',
          'on-success': '0 0% 100%',
          'success-container': '140 70% 90%',
          'on-success-container': '140 70% 10%',

          warning: '40 95% 50%',
          'on-warning': '0 0% 10%',
          'warning-container': '40 95% 90%',
          'on-warning-container': '40 95% 10%',

          error: '0 80% 50%',
          'on-error': '0 0% 100%',
          'error-container': '0 80% 90%',
          'on-error-container': '0 80% 10%',
        }
      ],
      defaultTheme: 'brand',
    }),
  ],
}
```

### TypeScript Custom Theme

For type safety, use TypeScript:

```typescript
// tailwind.config.ts
import { duskmoonui, type ThemeConfig } from '@duskmoon-dev/core';
import type { Config } from 'tailwindcss';

const brandTheme: ThemeConfig = {
  name: 'brand',
  primary: '220 90% 50%',
  'primary-focus': '220 90% 40%',
  // ... rest of theme tokens
};

const config: Config = {
  plugins: [
    duskmoonui({
      themes: ['sunshine', 'moonlight', brandTheme],
      defaultTheme: 'brand',
    }),
  ],
};

export default config;
```

## Understanding HSL Values

@duskmoon-dev/core uses HSL color format: `hue saturation% lightness%`

### Hue (0-360)

The color on the color wheel:

- **0° / 360°**: Red
- **60°**: Yellow
- **120°**: Green
- **180°**: Cyan
- **240°**: Blue
- **300°**: Magenta

### Saturation (0-100%)

Color intensity:

- **0%**: Grayscale
- **50%**: Muted
- **100%**: Vibrant

### Lightness (0-100%)

Brightness:

- **0%**: Black
- **50%**: Pure color
- **100%**: White

### Example Values

```javascript
{
  // Pure red at 50% lightness
  primary: '0 100% 50%',

  // Muted blue at 60% lightness
  secondary: '240 50% 60%',

  // Vibrant green at 45% lightness
  tertiary: '120 70% 45%',

  // Light gray surface
  surface: '0 0% 98%',
}
```

## Color Relationships

### Focus States

Focus states should be slightly darker (lower lightness):

```javascript
{
  primary: '220 90% 50%',        // Base color
  'primary-focus': '220 90% 40%', // 10% darker
}
```

### Container Colors

Container colors should be much lighter for backgrounds:

```javascript
{
  primary: '220 90% 50%',              // Vibrant
  'primary-container': '220 90% 90%',  // Very light
}
```

### On-Color Text

Text on colored backgrounds:

```javascript
{
  primary: '220 90% 50%',             // Dark blue background
  'primary-content': '0 0% 100%',     // White text

  'primary-container': '220 90% 90%', // Light blue background
  'on-primary-container': '220 90% 10%', // Dark blue text
}
```

## Theme Pairs (Light + Dark)

Create matching light and dark themes:

```javascript
const lightTheme = {
  name: 'brand-light',
  primary: '220 90% 50%',
  secondary: '280 70% 55%',
  surface: '0 0% 100%',        // White
  'on-surface': '0 0% 10%',    // Near black
};

const darkTheme = {
  name: 'brand-dark',
  primary: '220 90% 60%',      // Slightly lighter for dark mode
  secondary: '280 70% 65%',    // Slightly lighter
  surface: '0 0% 10%',         // Near black
  'on-surface': '0 0% 90%',    // Near white
};

module.exports = {
  plugins: [
    duskmoonui({
      themes: [lightTheme, darkTheme],
      defaultTheme: 'brand-light',
    }),
  ],
};
```

## Using Brand Colors

### Extract Colors from Brand Guidelines

If you have hex colors from brand guidelines, convert to HSL:

```javascript
// Brand colors (hex)
const brandColors = {
  blue: '#2563EB',     // Convert to HSL: 220 90% 65%
  purple: '#9333EA',   // Convert to HSL: 280 85% 55%
  teal: '#14B8A6',     // Convert to HSL: 172 76% 40%
};

// Use in theme
{
  primary: '220 90% 65%',    // Brand blue
  secondary: '280 85% 55%',  // Brand purple
  tertiary: '172 76% 40%',   // Brand teal
}
```

### Online HSL Converter

Use tools like [HSL Color Picker](https://hslpicker.com/) to convert colors.

## Surface Elevation System

Create a harmonious elevation system for surfaces:

```javascript
{
  // Base surface
  surface: '0 0% 100%',

  // Elevation levels (progressively darker)
  'surface-dim': '0 0% 87%',              // Dimmed
  'surface-bright': '0 0% 100%',          // Bright

  'surface-container-lowest': '0 0% 100%', // Level 0
  'surface-container-low': '0 0% 96%',     // Level 1
  'surface-container': '0 0% 94%',         // Level 2
  'surface-container-high': '0 0% 92%',    // Level 3
  'surface-container-highest': '0 0% 90%', // Level 4

  // Text colors
  'on-surface': '0 0% 10%',               // Primary text
  'on-surface-variant': '0 0% 30%',       // Secondary text
}
```

For dark themes, reverse the lightness values:

```javascript
{
  surface: '0 0% 10%',                     // Dark base

  'surface-container-lowest': '0 0% 10%',  // Level 0
  'surface-container-low': '0 0% 14%',     // Level 1
  'surface-container': '0 0% 16%',         // Level 2
  'surface-container-high': '0 0% 18%',    // Level 3
  'surface-container-highest': '0 0% 20%', // Level 4

  'on-surface': '0 0% 90%',                // Light text
}
```

## Accessibility

### Contrast Ratios

Ensure WCAG AA compliance (4.5:1 for normal text):

```javascript
// Good contrast
{
  primary: '220 90% 50%',          // Dark blue
  'primary-content': '0 0% 100%',  // White (high contrast)
}

// Poor contrast (avoid)
{
  primary: '220 90% 70%',          // Light blue
  'primary-content': '0 0% 100%',  // White (low contrast)
}
```

### Testing Contrast

Use tools like:
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
- [Coolors Contrast Checker](https://coolors.co/contrast-checker)
- Browser DevTools accessibility panel

### Color Blindness

Test your theme with color blindness simulators:
- [Coblis Color Blindness Simulator](https://www.color-blindness.com/coblis-color-blindness-simulator/)
- Use patterns and icons in addition to color

## Multiple Custom Themes

Support multiple themes for different brands or contexts:

```javascript
const corporateTheme = {
  name: 'corporate',
  primary: '220 90% 50%',    // Professional blue
  secondary: '0 0% 40%',     // Gray
  tertiary: '200 70% 50%',   // Light blue
  // ...
};

const playfulTheme = {
  name: 'playful',
  primary: '330 80% 60%',    // Pink
  secondary: '45 95% 55%',   // Yellow
  tertiary: '260 70% 60%',   // Purple
  // ...
};

module.exports = {
  plugins: [
    duskmoonui({
      themes: ['sunshine', 'moonlight', corporateTheme, playfulTheme],
      defaultTheme: 'corporate',
    }),
  ],
};
```

Switch between themes:

```javascript
document.documentElement.setAttribute('data-theme', 'playful');
```

## Best Practices

### Start with One Brand Color

Build your theme from a single brand color:

```javascript
const brandHue = 220; // Your brand color hue

{
  // Primary uses brand hue
  primary: `${brandHue} 90% 50%`,

  // Secondary shifts hue by 60 degrees
  secondary: `${brandHue + 60} 70% 55%`,

  // Tertiary shifts by -60 degrees
  tertiary: `${brandHue - 60} 70% 45%`,
}
```

### Maintain Consistency

Keep saturation and lightness values consistent within color groups:

```javascript
// Good: Consistent saturation/lightness
{
  primary: '220 90% 50%',
  secondary: '280 90% 50%',    // Same saturation/lightness
  tertiary: '160 90% 50%',     // Same saturation/lightness
}

// Inconsistent: Hard to harmonize
{
  primary: '220 90% 50%',
  secondary: '280 50% 70%',    // Different saturation/lightness
  tertiary: '160 70% 30%',     // Different saturation/lightness
}
```

### Test in Context

Always test your theme with real components:

```html
<div data-theme="brand">
  <button class="btn btn-primary">Primary Button</button>
  <div class="card card-default">
    <div class="card-body">
      <h3>Card Title</h3>
      <p>Card content</p>
    </div>
  </div>
  <div class="alert alert-success">Success message</div>
</div>
```

## Exporting Themes

Share themes as npm packages:

```javascript
// my-theme/index.js
module.exports = {
  brandLight: {
    name: 'brand-light',
    // ... theme tokens
  },
  brandDark: {
    name: 'brand-dark',
    // ... theme tokens
  },
};

// Usage in consumer project
const { brandLight, brandDark } = require('my-theme');

module.exports = {
  plugins: [
    duskmoonui({
      themes: [brandLight, brandDark],
    }),
  ],
};
```
