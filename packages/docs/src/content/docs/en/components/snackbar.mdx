# Snackbar

Snackbars provide brief messages about app processes at the bottom of the screen.

## Basic Usage

```html
<div class="snackbar snackbar-bottom snackbar-show">
  <span class="snackbar-message">Message sent</span>
</div>
```

## JavaScript Example

```javascript
function showSnackbar(message, duration = 3000) {
  const snackbar = document.createElement('div');
  snackbar.className = 'snackbar snackbar-bottom';
  snackbar.innerHTML = `<span class="snackbar-message">${message}</span>`;

  document.body.appendChild(snackbar);

  // Trigger animation
  setTimeout(() => snackbar.classList.add('snackbar-show'), 10);

  // Auto hide
  setTimeout(() => {
    snackbar.classList.remove('snackbar-show');
    setTimeout(() => snackbar.remove(), 300);
  }, duration);
}

// Usage
showSnackbar('Message sent successfully');
```
