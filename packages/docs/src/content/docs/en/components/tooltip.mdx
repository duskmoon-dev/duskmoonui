## JavaScript Example

```javascript
// Show tooltip on hover
document.querySelectorAll('[data-tooltip]').forEach(element => {
  const tooltipText = element.getAttribute('data-tooltip');
  const tooltip = document.createElement('div');
  tooltip.className = 'tooltip tooltip-top';
  tooltip.innerHTML = `${tooltipText}<div class="tooltip-arrow"></div>`;
  element.style.position = 'relative';
  element.appendChild(tooltip);

  element.addEventListener('mouseenter', () => {
    tooltip.classList.add('tooltip-show');
  });

  element.addEventListener('mouseleave', () => {
    tooltip.classList.remove('tooltip-show');
  });
});
```

## API Reference

| Class | Description |
|-------|-------------|
| `.tooltip` | Base tooltip (required) |
| `.tooltip-show` | Show tooltip |
| `.tooltip-top` | Position above element |
| `.tooltip-bottom` | Position below element |
| `.tooltip-left` | Position to the left |
| `.tooltip-right` | Position to the right |
| `.tooltip-arrow` | Arrow element |
| `.tooltip-no-arrow` | Hide arrow |
| `.tooltip-sm` | Small size |
| `.tooltip-lg` | Large size |
| `.tooltip-rich` | Rich content tooltip |
