# Flower Animation

An animated garden scene rendered entirely in CSS. Four flowers bloom and sway on their stems, surrounded by multiple layers of swaying grass. Glowing light particles rise from the flower centers, and heart-shaped bubbles float upward across the dark starlit sky. All motion — blooming, growing, swaying, and floating — is driven by CSS `@keyframes` and `animation-delay` staggering alone, with no JavaScript.

Original art: "Flores y corazones" — ported from SCSS to plain CSS and adapted to the css-art component format.
