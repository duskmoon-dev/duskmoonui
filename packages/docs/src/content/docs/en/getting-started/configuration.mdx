# Configuration

@duskmoon-dev/core is highly configurable. This guide covers all available options to customize the plugin for your needs.

## Basic Configuration

The simplest configuration uses default settings with Tailwind CSS 4:

```css
@import "tailwindcss";
@plugin "@duskmoon-dev/core";
```

**Note:** @duskmoon-dev/core requires Tailwind CSS 4.0 or later. Tailwind CSS 3.x is not supported.

## Plugin Options

Customize @duskmoon-dev/core by passing options in Tailwind CSS 4:

```css
@import "tailwindcss";
@plugin "@duskmoon-dev/core" {
  @theme {
    --theme: "sunshine";
    --dark-theme: "moonlight";
  }
  @config {
    components: ["button", "card", "input"];
    styled: true;
    prefix: "";
  }
}
```

## Available Options

### `themes`

**Type:** `string[]`
**Default:** `['sunshine', 'moonlight']`

Array of theme names to include. Built-in themes are `sunshine` (light) and `moonlight` (dark).

```javascript
{
  themes: ['sunshine', 'moonlight']
}
```

You can also provide custom theme objects:

```javascript
import { duskmoonui } from '@duskmoon-dev/core';

{
  themes: [
    'sunshine',
    {
      name: 'custom',
      primary: '280 80% 55%',      // Purple
      secondary: '200 80% 55%',    // Blue
      tertiary: '160 75% 45%',     // Teal
      // ... other tokens
    }
  ]
}
```

### `defaultTheme`

**Type:** `string`
**Default:** `'sunshine'`

The theme to use when no `data-theme` attribute is present.

```javascript
{
  defaultTheme: 'moonlight'
}
```

### `styled`

**Type:** `boolean`
**Default:** `true`

Whether to include component styles. Set to `false` if you only want theme CSS variables.

```javascript
{
  styled: false  // Only inject theme variables, no component styles
}
```

### `components`

**Type:** `'all' | string[]`
**Default:** `'all'`

Which components to include. Use `'all'` for everything, or specify an array:

```javascript
{
  components: ['button', 'card', 'input']  // Only these components
}
```

Available components:
- `button` - Button variants and icon buttons
- `card` - Card container with subcomponents
- `input` - Form inputs, textarea, select, checkbox, radio
- `badge` - Badge indicators and notifications
- `alert` - Alert messages and notifications

### `prefix`

**Type:** `string`
**Default:** `''`

Prefix for component classes. Useful to avoid conflicts:

```javascript
{
  prefix: 'dm-'  // Classes become .dm-btn, .dm-card, etc.
}
```

Example usage with prefix:

```html
<button class="dm-btn dm-btn-primary">Button</button>
```

## Theme Customization

### Custom Theme Colors

Create custom themes by extending or overriding the color palette:

```javascript
import { duskmoonui, type ThemeConfig } from '@duskmoon-dev/core';

const customTheme: ThemeConfig = {
  name: 'ocean',

  // Primary colors (main brand color)
  primary: '200 90% 50%',           // Ocean blue
  'primary-focus': '200 90% 40%',
  'primary-content': '0 0% 100%',
  'primary-container': '200 90% 90%',
  'on-primary-container': '200 90% 10%',

  // Secondary colors (accent color)
  secondary: '280 70% 55%',         // Purple
  'secondary-focus': '280 70% 45%',
  'secondary-content': '0 0% 100%',
  'secondary-container': '280 70% 90%',
  'on-secondary-container': '280 70% 10%',

  // Tertiary colors (third brand color)
  tertiary: '160 70% 45%',          // Teal
  'tertiary-focus': '160 70% 35%',
  'tertiary-content': '0 0% 100%',
  'tertiary-container': '160 70% 90%',
  'on-tertiary-container': '160 70% 10%',

  // Surface colors
  surface: '0 0% 100%',
  'surface-dim': '0 0% 87%',
  'surface-bright': '0 0% 100%',
  'surface-container-lowest': '0 0% 100%',
  'surface-container-low': '0 0% 96%',
  'surface-container': '0 0% 94%',
  'surface-container-high': '0 0% 92%',
  'surface-container-highest': '0 0% 90%',
  'on-surface': '0 0% 10%',
  'on-surface-variant': '0 0% 30%',
  'surface-variant': '200 20% 90%',

  // Outline colors
  outline: '0 0% 50%',
  'outline-variant': '0 0% 80%',

  // Semantic colors
  info: '200 90% 50%',
  'on-info': '0 0% 100%',
  'info-container': '200 90% 90%',
  'on-info-container': '200 90% 10%',

  success: '140 70% 45%',
  'on-success': '0 0% 100%',
  'success-container': '140 70% 90%',
  'on-success-container': '140 70% 10%',

  warning: '40 95% 50%',
  'on-warning': '0 0% 10%',
  'warning-container': '40 95% 90%',
  'on-warning-container': '40 95% 10%',

  error: '0 80% 50%',
  'on-error': '0 0% 100%',
  'error-container': '0 80% 90%',
  'on-error-container': '0 80% 10%',
};

module.exports = {
  plugins: [
    duskmoonui({
      themes: ['sunshine', 'moonlight', customTheme],
      defaultTheme: 'ocean',
    }),
  ],
}
```

### HSL Color Format

All colors use the HSL format without the `hsl()` wrapper:

```javascript
{
  primary: '200 90% 50%'  // hue saturation% lightness%
}
```

This format allows Tailwind to apply opacity utilities:

```html
<div class="bg-primary bg-opacity-50">  <!-- 50% opacity -->
```

## Component Customization

### Selective Component Loading

Load only the components you need for smaller bundle sizes:

```javascript
{
  components: ['button', 'input']  // Only button and input components
}
```

### Disable Component Styles

If you want to use only the theme system:

```javascript
{
  styled: false,  // No component styles, only theme CSS variables
}
```

Then use theme variables in your custom components:

```css
.my-button {
  background-color: hsl(var(--color-primary));
  color: hsl(var(--color-primary-content));
}
```

## Complete Example

Here's a complete configuration example:

```javascript
// tailwind.config.js
const { duskmoonui } = require('@duskmoon-dev/core');

module.exports = {
  content: [
    './src/**/*.{html,js,jsx,ts,tsx,vue,astro}',
  ],
  plugins: [
    duskmoonui({
      // Use built-in and custom themes
      themes: [
        'sunshine',
        'moonlight',
        {
          name: 'brand',
          primary: '220 90% 50%',
          secondary: '280 70% 55%',
          tertiary: '160 70% 45%',
          // ... other tokens
        }
      ],

      // Default theme
      defaultTheme: 'sunshine',

      // Include component styles
      styled: true,

      // Load only needed components
      components: ['button', 'card', 'input', 'badge'],

      // Optional class prefix
      prefix: '',
    }),
  ],
}
```

## TypeScript Support

@duskmoon-dev/core includes full TypeScript definitions:

```typescript
import { duskmoonui, type ThemeConfig, type DuskMoonUIOptions } from '@duskmoon-dev/core';

const options: Partial<DuskMoonUIOptions> = {
  themes: ['sunshine', 'moonlight'],
  defaultTheme: 'sunshine',
  styled: true,
  components: 'all',
};
```

## Environment Variables

You can use environment variables in your configuration:

```javascript
{
  defaultTheme: process.env.DEFAULT_THEME || 'sunshine',
  styled: process.env.NODE_ENV !== 'test',
}
```
