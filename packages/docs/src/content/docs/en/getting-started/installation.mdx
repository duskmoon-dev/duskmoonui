# Installation

Get started with DuskMoonUI in just a few steps.

## Prerequisites

Before installing DuskMoonUI, ensure you have:

- Node.js 18+ or Bun 1.0+
- Tailwind CSS 4.0+

## Install with Bun (Recommended)

```bash
bun add duskmoonui
```

## Install with npm

```bash
npm install duskmoonui
```

## Install with pnpm

```bash
pnpm add duskmoonui
```

## Configure Tailwind CSS

Add DuskMoonUI to your Tailwind configuration:

### Using Tailwind CSS 4.0+

In your `globals.css` or main CSS file:

```css
@import "tailwindcss";
@plugin "duskmoonui";
```

### Using Tailwind CSS 3.x

In your `tailwind.config.js`:

```javascript
module.exports = {
  plugins: [
    require('duskmoonui'),
  ],
}
```
